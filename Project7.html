    웹 기술의 등장 과정을 조사하여 설명하시오.



    과제 목표



    웹기술의 등장 과정:
    1) 어느 단체의 누가 어떤 목적으로 웹 기술을 만들었는지?
    2) 웹 기술의 처음 목적과 지금 활용되는 상황이 어떻게 다른지?

    1) 어느 단체의 누가 어떤 목적으로 웹 기술을 만들었는지?


    팀 버너스 리와 그의 팀은 CERN에서 HTML뿐 아니라
    웹 브라우저 및 텍스트 기반 웹 브라우저 관련 기술과 더불어 오리지널 HTTP을 발명하였다.
    웹 브라우저 같은 응용 프로그램을 통해 클라이언트와 서버 사이 데이터를 전송하는 프로토콜(통신규약)을 통해 분산
    되어 있는 서버와 사용자 간에 하이퍼텍스트를 이용한 정보 교환이 가능하게 하기위한 목적으로 만들어졌다.

    2) 웹 기술의 처음 목적과 지금 활용되는 상황이 어떻게 다른지?


    처음의 웹 기술은 운영자가 보여주는것 이외에는 접할 수 없었고 동적인
    데이터를 제공하는 서비스도 없었다. 하드디스크의 저장공간도 충분하지않고
    네트워크의 대역폭도 작았으므로 도영상이나 플래시같은 동적이고 현란한
    웹사이트는 리소스를 낭비하는 것으로 여겨졌다.
    - 단순한 문서의 구조를 표현
    -텍스트와 그들의 링크가 주된 링크형태

    지금의 웹 기술은 개인화, 지능화, 상황인식 등으로 수렴되며, 시맥틱 기술을
    이용해서 상황인식을 통해 웹 이용자에게 맞춤형 콘텐츠 및 서비스를 제공해주고있다
    -다양한 기술을 제공해주는 언어들이 등장했다.
    -Semantic Web 페이지를 태그를 통해서 각각의 의미를 부여할 수 있는 기능 제공
    -HTML5 + Javacript + CSS3만으로 프로그램 구현가능하다.
    -복잡해진 구조의 웹이 등장하면서 성능 뿐만 아니라 보안도 강화되었다.
    -소극적인 자세에서 적극적으로 자신들만의 웹 콘텐츠를 만들어 나가는 자세로 변했다.
